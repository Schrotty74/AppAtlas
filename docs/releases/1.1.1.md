AppAtlas 1.1.1 is a stable bugfix release.

## Fixed

- Catalog exports with license data now read only known license records from
  the macOS Keychain instead of checking every app in the catalog.
- Removed an additional Keychain authentication context that could trigger
  unnecessary macOS password dialogs during export.
- Export data is now created only after the save location has been selected,
  avoiding overlapping app sheets and Keychain dialogs.
- Catalog import uses the native macOS file picker reliably again.
- Local development builds no longer run the privacy audit. Privacy checks
  remain active for commit, push, release and backup workflows.

## Privacy

AppAtlas still starts without a personal catalog. Catalogs, local paths,
license values, user-specific icons and backup files are not included in the
source code or release package.

License values remain in the macOS Keychain and are exported only after the
user explicitly chooses an export option that includes license data.
